Tipos de Dados em JavaScript

JavaScript possui vários tipos de dados, os principais são:

  • Number: Representa números, tanto inteiros quanto decimais.
  • String: Representa sequências de caracteres, ou seja, texto.
  • Boolean: Representa valores lógicos, podendo ser verdadeiro (true) ou falso (false).
  • Object: Representa coleções de dados e funcionalidades.
  • Array: É um tipo especial de objeto que armazena uma lista ordenada de valores.
  • null: Representa a ausência intencional de qualquer valor.
  • undefined: Indica que uma variável foi declarada mas não foi atribuída a nenhum valor.

Exemplos:

                
                    // Exemplo de Number
                    let idade = 30; // Número inteiro
                    let altura = 1.75; // Número decimal

                    // Exemplo de String
                    let nome = "João"; // Texto entre aspas
                    let cidade = 'São Paulo'; //
                    // Exemplo de Boolean
                    let isAtivo = true; // Verdadeiro
                    let isAdmin = false; // Falso
                    // Exemplo de Object
                    let pessoa = {
                        nome: "Maria",
                        idade: 25,
                        cidade: "Rio de Janeiro"
                    };
                    // Exemplo de Array
                    let frutas = ["maçã", "banana", "laranja"]; // Lista de frutas
                    // Exemplo de null
                    let carro = null; // Ausência de valor
                    // Exemplo de undefined
                    let endereco; // Variável declarada, mas sem valor atribuído
